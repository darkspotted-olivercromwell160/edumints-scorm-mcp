# edumints SCORM MCP

> **Интерактивті, SCORM-үйлесімді электрондық оқыту курстарын құрастыратын MCP сервері.**
> Сіз (немесе Claude сияқты ЖИ-клиент) — **автор**; бұл сервер — **құрастырушы**.
> Курсты құрылымдалған спецификация ретінде сипаттаңыз — сервер тексеріп, рендерлеп, кез келген
> LMS-те (Moodle, SCORM Cloud, …) жұмыс істейтін **дербес SCORM zip** етіп бумалайды.

**🌐 Тілдер:** [English](README.md) · [Türkçe](README.tr.md) · [Español](README.es.md) · [Русский](README.ru.md) · [简体中文](README.zh-CN.md) · [Azərbaycanca](README.az.md) · [Қазақша](README.kk.md) · [Кыргызча](README.ky.md)

Ашық бастапқы код, **[edumints.com](https://edumints.com)** платформасы әзірлеген. **Өз
компьютеріңізде немесе серверіңізде хостинг** үшін жасалған және **үлес қосуға ашық**.

---

## Идея (басқаша тәсіл)

Электрондық оқытудың көбі ауыр үстелдік құралдармен қолмен жасалады. Мұнда **ЖИ-клиент курсты
сипаттайды** (мақсаттар, экрандар, тесттер, тармақталу, медиа) — [Model Context Protocol](https://modelcontextprotocol.io)
арқылы, ал сервер ең қиынын істейді: тексеру, премиум тема, қолжетімді HTML рендеринг, SCORM runtime
көпірі және бумалау. Нәтиже — стандарттарға сай SCORM бумасы, провайдерге тәуелділіксіз.

**Автор = MCP клиенті · Құрастырушы = бұл сервер.**

## Мүмкіндіктер

- **18+ экран түрі** — тақырып, мазмұн, көп таңдау, дұрыс/бұрыс, бос орынды толтыру, сүйреп апару,
  hotspot, тармақталатын сценарий, аккордеон, қойындылар, карточкалар, сәйкестендіру, сұрыптау,
  уақыт сызығы, lottie, **басқарылатын бағдарлама симуляциясы**, видео, қорытынды.
- **Слайд-сахна ойнатқышы** — кез келген экранға масштабталатын тұрақты 16:9 сахна, ойнату панелі
  (ойнату/іздеу/субтитр/мәзір/қайталау) және дыбыстаумен синхронды **уақыт сызығы бойынша көрсету**.
  Бөлімдерге топталған мәзір. Реттелетін сахна өлшемі; толық бейімделгіш/мобильді; ендірілген SVG
  белгішелер (эмодзи жоқ).
- **Логика және ойынтехникасы** — айнымалылар/күй, шартты көріну, тармақталу, ұпай мен таймер HUD-ы.
- **Бағалау** — мақсатқа сай сұрақтар, дұрыс/бұрыс үшін кері байланыс, SCORM-ға жазылатын балл.
- **Медиа** — MCP-аралық қабылдау (аудио/сурет/видеоны өз MCP-леріңізден әкеліңіз → `add_asset`),
  ffmpeg өңдеу, **бағдарламалық motion-graphic/деректер визуализациясы видеосы** (HyperFrames) және
  жылдам дыбыстау үшін ендірілген **түрік TTS** (Piper, офлайн).
- **Тема және қолжетімділік** — ашық/бейтарап/жоғары контраст пресеттер, бренд токендері, WCAG-ге
  бағдарланған, `prefers-reduced-motion` қолдауы.
- **SCORM 1.2 және 2004**, детерминделген бумалау, шығын шектеулері, опт-ин/жалқау ауыр функциялар
  (курс пайдаланбаса жүктелмейді).

## Жылдам бастау (өзіңіз хостинг)

### Docker (ұсынылады)
```bash
git clone https://github.com/kemalyy/edumints-scorm-mcp.git
cd edumints-scorm-mcp
docker build -t edumints-scorm-mcp .
docker run -p 8000:8000 -v "$PWD/data:/data" edumints-scorm-mcp
# MCP нүктесі: http://localhost:8000/mcp   ·   денсаулық: http://localhost:8000/health
```
Имидж барлық қосымша функциялар үшін қажеттінің бәрін қамтиды (ffmpeg, видео үшін Node + HyperFrames,
TTS үшін Piper + түрік даусы).

### Жергілікті (Python)
```bash
python -m venv .venv && source .venv/bin/activate
pip install ".[tts]"          # ".[tts]" офлайн түрік TTS-ін (Piper) қосады; қажет болмаса алып тастаңыз
python server.py              # MCP-ді HTTP арқылы ұсынады
```
Видео жасау үшін Node 22+ және HyperFrames (`npm i -g hyperframes`) + ffmpeg орнатыңыз.

### Конфигурация
`.env.example`-ті көшіріп баптаңыз (деректер қалтасы, квоталар, базалық URL, TTL). Барлық параметрлер
файлда. Жергілікті іске қосу үшін құпия деректер қажет емес.

## ЖИ-клиентті қосу

Кез келген MCP клиентін `http://<хост>:8000/mcp` мекенжайына бағыттаңыз:
- **Claude** (десктоп/веб/Code) — коннектор / MCP сервер ретінде қосыңыз.
- **Antigravity** және басқа MCP клиенттер — сол нүкте (HTTP/Streamable).

Содан кейін сұраңыз: *«X тақырыбында тесті мен қорытындысы бар 6 минуттық интерактивті курс жаса.»*
Клиент төмендегі құралдарды шақырады; сіз жүктеуге болатын SCORM zip аласыз.

> **Авторлық skill-мен** бірге жұмыс істейді (осы сервермен сапалы курс жасауды ЖИ-клиентке үйрететін
> Claude Agent Skill): https://github.com/kemalyy/edumints-scorm-skill

## Негізгі құралдар (MCP)

| Құрал | Мақсаты |
|---|---|
| `build_from_spec` | Бір JSON спецификация → тексерілген жоба + бумаланған SCORM zip (негізгі жол) |
| `create_project` / `add_screen` / `update_screen` / … | Ұсақ, қадамдық өңдеу |
| `set_theme` / `set_tracking` | Тема + аяқтау/бағалау ережелері |
| `add_asset` | Аудио/сурет/видео қабылдау (data-URI немесе https, SSRF-қорғалған) |
| `synthesize_speech` | Ендірілген түрік дыбыстауы (Piper, офлайн) → аудио ресурс |
| `make_video_from_image_audio` / `render_motion_video` / `render_screen_video` | Видео (ffmpeg / HyperFrames) |
| `preview` / `validate_package` / `build_package` | Алдын ала қарау, тексеру, SCORM zip жүктеу |

## Архитектура

```
MCP клиенті (автор)  ──►  scorm-mcp (құрастырушы)
                          ├─ core/        модельдер (Pydantic), бумалау, сақтау
                          ├─ components/  HTML рендерер + runtime қозғалтқышы + видео компиляторы
                          ├─ auth/        API-кілт + OAuth, SSRF қорғаныс
                          ├─ themes/      дизайн токендері / пресеттер
                          ├─ runtime/     ендірілген SCORM runtime (scorm-again, MIT)
                          └─ server.py    FastMCP құралдары (HTTP)
```
Нәтиже: дербес `index.html` + `imsmanifest.xml` + ресурстар + SCORM runtime, zip-телген.
