# edumints SCORM MCP

> **Интерактивдүү, SCORM-шайкеш онлайн окуу курстарын чогултуучу MCP сервери.**
> Сиз (же Claude сыяктуу ЖИ-кардар) — **автор**; бул сервер — **чогултуучу**.
> Курсту структураланган спецификация катары сүрөттөңүз — сервер текшерет, рендерлейт жана каалаган
> LMS-те (Moodle, SCORM Cloud, …) иштеген **өз алдынча SCORM zip** катары таңгактайт.

**🌐 Тилдер:** [English](README.md) · [Türkçe](README.tr.md) · [Español](README.es.md) · [Русский](README.ru.md) · [简体中文](README.zh-CN.md) · [Azərbaycanca](README.az.md) · [Қазақша](README.kk.md) · [Кыргызча](README.ky.md)

Ачык булак, **[edumints.com](https://edumints.com)** платформасы тарабынан иштелип чыккан. **Өз
компьютериңизде же серверыңизде хостинг кылуу** үчүн ылайыкталган жана **салым кошууга ачык**.

---

## Идея (башкача мамиле)

Онлайн окутуунун көбү оор иштел столу куралдары менен кол менен курулат. Бул жерде **ЖИ-кардар курсту
сүрөттөйт** (максаттар, экрандар, тесттер, тармакталуу, медиа) — [Model Context Protocol](https://modelcontextprotocol.io)
аркылуу, ал эми сервер эң кыйынын аткарат: текшерүү, премиум тема, жеткиликтүү HTML рендеринг, SCORM
runtime көпүрөсү жана таңгактоо. Жыйынтыгы — стандарттарга шайкеш SCORM пакети, провайдерге көз
карандысыз.

**Автор = MCP кардары · Чогултуучу = бул сервер.**

## Мүмкүнчүлүктөр

- **18+ экран түрү** — аталыш, мазмун, көп тандоо, туура/туура эмес, бош орунду толтуруу, сүйрөп коюу,
  hotspot, тармакталган сценарий, аккордеон, өтмөктөр, карточкалар, дал келтирүү, иреттөө, убакыт
  сызыгы, lottie, **жетектелген программа симуляциясы**, видео, корутунду.
- **Слайд-сахна ойноткучу** — каалаган экранга масштабталган туруктуу 16:9 сахна, ойнотуу панели
  (ойнотуу/издөө/субтитр/меню/кайталоо) жана үндөө менен синхрондуу **убакыт сызыгы боюнча көрсөтүү**.
  Бөлүмдөргө топтоштурулган меню. Жөнгө салынуучу сахна өлчөмү; толук ыкчам/мобилдик; камтылган SVG
  белгичелер (эмодзи жок).
- **Логика жана оюндаштыруу** — өзгөрмөлөр/абал, шарттуу көрүнүү, тармакталуу, упай жана таймер HUD-у.
- **Баалоо** — максатка ылайык суроолор, туура/туура эмес үчүн кайтарым байланыш, SCORM-го жазылган упай.
- **Медиа** — MCP-аралык кабыл алуу (аудио/сүрөт/видеону өз MCP-лериңизден алып келиңиз → `add_asset`),
  ffmpeg иштетүү, **программалык motion-graphic/маалымат визуализациясы видеосу** (HyperFrames) жана
  ылдам үндөө үчүн камтылган **түрк TTS** (Piper, оффлайн).
- **Тема жана жеткиликтүүлүк** — ачык/нейтралдуу/жогорку контраст пресеттер, бренд токендери, WCAG-ге
  багытталган, `prefers-reduced-motion` колдоосу.
- **SCORM 1.2 жана 2004**, детерминделген таңгактоо, чыгым чектөөлөрү, опт-ин/жалкоо оор функциялар
  (курс колдонбосо жүктөлбөйт).

## Тез баштоо (өзүңүз хостинг)

### Docker (сунушталат)
```bash
git clone https://github.com/kemalyy/edumints-scorm-mcp.git
cd edumints-scorm-mcp
docker build -t edumints-scorm-mcp .
docker run -p 8000:8000 -v "$PWD/data:/data" edumints-scorm-mcp
# MCP чекити: http://localhost:8000/mcp   ·   ден соолук: http://localhost:8000/health
```
Образ бардык кошумча функциялар үчүн керектүүнүн баарын камтыйт (ffmpeg, видео үчүн Node + HyperFrames,
TTS үчүн Piper + түрк үнү).

### Жергиликтүү (Python)
```bash
python -m venv .venv && source .venv/bin/activate
pip install ".[tts]"          # ".[tts]" оффлайн түрк TTS-ин (Piper) кошот; керек болбосо алып салыңыз
python server.py              # MCP-ди HTTP аркылуу сунуштайт
```
Видео жаратуу үчүн Node 22+ жана HyperFrames (`npm i -g hyperframes`) + ffmpeg орнотуңуз.

### Конфигурация
`.env.example`-ти көчүрүп ыңгайлаштырыңыз (маалымат папкасы, квоталар, базалык URL, TTL). Бардык
параметрлер файлда. Жергиликтүү иштетүү үчүн купуя маалымат талап кылынбайт.

## ЖИ-кардарды туташтыруу

Каалаган MCP кардарын `http://<хост>:8000/mcp` дарегине багыттаңыз:
- **Claude** (десктоп/веб/Code) — коннектор / MCP сервер катары кошуңуз.
- **Antigravity** жана башка MCP кардарлар — ошол эле чекит (HTTP/Streamable).

Андан соң сураңыз: *«X темасында тести жана корутундусу бар 6 мүнөттүк интерактивдүү курс жаса.»*
Кардар төмөнкү куралдарды чакырат; сиз жүктөп алса боло турган SCORM zip аласыз.

> **Авторлук skill менен** бирге иштейт (ушул сервер менен сапаттуу курс жасоону ЖИ-кардарга үйрөткөн
> Claude Agent Skill): https://github.com/kemalyy/edumints-scorm-skill

## Негизги куралдар (MCP)

| Курал | Максаты |
|---|---|
| `build_from_spec` | Бир JSON спецификация → текшерилген долбоор + таңгакталган SCORM zip (негизги жол) |
| `create_project` / `add_screen` / `update_screen` / … | Майда, кадамдык түзөтүү |
| `set_theme` / `set_tracking` | Тема + аяктоо/баалоо эрежелери |
| `add_asset` | Аудио/сүрөт/видео кабыл алуу (data-URI же https, SSRF-корголгон) |
| `synthesize_speech` | Камтылган түрк үндөөсү (Piper, оффлайн) → аудио ресурс |
| `make_video_from_image_audio` / `render_motion_video` / `render_screen_video` | Видео (ffmpeg / HyperFrames) |
| `preview` / `validate_package` / `build_package` | Алдын ала көрүү, текшерүү, SCORM zip жүктөө |

## Архитектура

```
MCP кардары (автор)  ──►  scorm-mcp (чогултуучу)
                          ├─ core/        моделдер (Pydantic), таңгактоо, сактоо
                          ├─ components/  HTML рендерер + runtime кыймылдаткычы + видео компилятору
                          ├─ auth/        API-ачкыч + OAuth, SSRF коргоо
                          ├─ themes/      дизайн токендери / пресеттер
                          ├─ runtime/     камтылган SCORM runtime (scorm-again, MIT)
                          └─ server.py    FastMCP куралдары (HTTP)
```
Жыйынтык: өз алдынча `index.html` + `imsmanifest.xml` + ресурстар + SCORM runtime, zip-телген.
